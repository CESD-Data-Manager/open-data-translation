# Open Data Translation

Tools, templates, and guidance for translating open data products between English and French for publication and reporting purposes.

This repository is maintained by the Coastal Ecosystems Science Division (CESD) to support the preparation of bilingual datasets, metadata, data dictionaries, and supporting documentation.

## Purpose

The objective of this repository is to provide reusable resources that help staff:

- create bilingual metadata;
- translate data dictionaries;
- review machine-generated translations;
- maintain consistency across publications;
- support Government of Canada bilingual requirements;
- prepare datasets for publication on Open Government, OBIS, and related platforms.
